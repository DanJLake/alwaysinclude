alwaysInclude

<img>

Image tags must always contain the following:

  • src="" - The src of an image is its source file.
  • alt="" - The alt value of an image is used when an image cannot be displayed. This could be read by a screen reader, or displayed on screen due to an invalid src.

<label>

Label tags must always contain the following:

  • for="" - The 'for' attribute on a label is used by screen readers to associate that label with a form control.
